Configuración de IntroDB Skip

Modo STRM/API: Este plugin busca marcadores de intro en bases de datos externas (IntroHater e IntroDB) usando metadatos (IMDb ID) para evitar el escaneo pesado de archivos locales.
Activar la sincronización automática de intros.
Servidor primario (por defecto: https://introhater.com)
Servidor secundario (por defecto: https://api.introdb.app)
Opcional: Tu clave para IntroDB (X-API-Key).
Frecuencia con la que se buscan nuevos intros en segundo plano.
Ignorar marcadores con puntuación baja (Recomendado: 0.75).
Reemplaza marcadores locales si la API tiene versiones más nuevas.
Busca el intro inmediatamente al abrir un video si no está en caché.
FUERZA el salto de intro desde el servidor para clientes Android que no muestran el botón nativo.
